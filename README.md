# 工事写真台帳

iPhone に保存済みの「黒板が焼き込まれた工事写真」を選び、並べ替え・施工区分の付与をして、
**写真3枚/ページの工事写真帳PDF**を生成し、メールで送付するための PWA アプリです。

公開URL: https://nsmtysak.github.io/koji-photo-app/

## 使い方（基本の流れ）

1. **工事情報**を入力（注文番号 / 工事名 / 工事場所）
2. **写真を選択／追加**（カメラロールから複数選択。黒板は焼き込み済み前提で無加工）
3. **並べ替え**（▲▼）・不要な写真は ✕ で削除（この順がPDF掲載順）
4. 各写真に**施工区分**を付与（候補チップ選択／自由入力。候補はその場で追加・削除可）
5. **PDFを生成** → プレビュー / 保存 / メールで送付
6. 別の工事を始めるときは **「最初からやり直す（クリア）」**

## 設定（右上「設定」）

- **自社情報**（表紙下部に表示）
- **メール雛形**（送信宛先 / 件名。件名は `{工事名}` を差し込み可）
- **施工区分タグ**の管理

設定・工事情報は localStorage に保存されます。**写真と写真ごとの区分はセッション中のみ**保持し、永続化しません。

## PDF について

- pdf-lib + fontkit を**ローカル同梱**（オフラインでも生成可）。
- 日本語フォントは **M PLUS 1p（TrueType）を全グリフ埋め込み**。
  - pdf-lib のサブセット機能はグリフ欠落の不具合があり、CFF（OpenType）フォントは
    埋め込みが壊れて iPhone で文字化けするため、TrueType＋全埋め込みにしています
    （その分フォント約1MBがPDFに含まれます）。
- 表紙（工事写真帳 / 注文番号・工事名・工事場所 / 自社情報）＋ 写真3枚/ページ（右に件名・場所・区分）。
- ファイル名: `工事写真帳_{工事名}_{日付}.pdf`

## メール送付の挙動と制約（重要）

ブラウザからのメール自動送信はできません。**Web Share API（共有シート）**でメールアプリに渡します。

- **iPhone（共有シート対応）**
  - 「メールで送付（PDF添付）」→ 共有シート → **メールを選ぶと PDF が添付**されます。
  - **件名**は雛形（`{工事名}`差し込み済み）を可能な範囲で渡します（iOS側の挙動により反映されない場合あり）。
  - **宛先は共有シートでは自動入力できません**。画面の「宛先」を**「コピー」**してメールの宛先に貼り付けてください。
- **共有シート非対応（PC等）**
  - 「メールで送付」で **PDFのダウンロード**と **mailto:（宛先・件名入り）**のメール作成画面が開きます。PDFは手動で添付してください。
- 「PDFを保存」: iPhone は共有シートの**「"ファイル"に保存」**で端末へ保存、PC等はダウンロード保存。

> iOS の制約上、「宛先・件名・添付」を1アクションで完全自動化することはできません。
> 添付（Web Share）＋宛先のワンタップコピーの組み合わせが最も確実なため、この方式にしています。

## 技術

- バニラ HTML / CSS / JavaScript（ビルドツール・フレームワーク不使用）
- PWA（manifest + Service Worker。SWは network-first で更新が反映されやすい）
- 外部ライブラリは PDF生成用（pdf-lib + fontkit）のみ

詳細な構成・方針は [CLAUDE.md](CLAUDE.md) を参照。
